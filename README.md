# Ju Rosas | NextJs App

Projeto pessoal desenvolvido em NextJs de venda de rosas do deserto

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Anthony-MPO-dev/Ju_Rosas-Site.git
```

Entre no diretório do projeto

```bash
  cd ju-rosas-site
```


Baixe as dependências next no diretório do projeto

```bash
  npm install next
```


Inicie o servidor

```bash
  npm run dev
```

Agora é só abrir o navegador e acessar: 
```bash
localhost:PORTA
```

## Aprendizados

Ao desenvolver este projeto eu aprendi como criar diretorios com NextJs e React, aprendi sobre typescript, adquiri alguns conhecimentos de tailwindcss e pude aperfeiçoar outros conhecimentos basicos de desenvolvimento web.
